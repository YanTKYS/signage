# iSignage

`isignage.html` だけで動作する、ローカル用の単一HTMLサイネージ作成・表示ツールです。PowerPointに近い感覚で、テキスト、画像、動画、フォルダ再生メディア、時計、図形を16:9キャンバスへ配置できます。

## 対応ブラウザ

- Microsoft Edge 最新版
- Google Chrome 最新版

File System Access APIを利用するため、FirefoxやSafariではフォルダ選択機能が利用できない場合があります。

## 起動方法

1. `isignage.html` をローカルPCに置きます。
2. EdgeまたはChromeで直接開きます。
3. Webサーバーや外部ライブラリは不要です。

## 基本操作

- 上部の「テキスト」「画像」「動画」「メディア」「図形」「時計」から追加します。
- オブジェクトはクリックで選択、ドラッグで移動できます。
- 選択枠の8つのハンドルでサイズ変更できます。
- `Delete` 削除、`Ctrl+C` コピー、`Ctrl+V` 貼り付け、`Ctrl+D` 複製、`Ctrl+Z` Undo、`Ctrl+Y` Redoに対応しています。
- 複数選択は `Ctrl` + クリック、`Ctrl+A` は現在シーン内の全選択です。
- 右側のプロパティ欄から位置、サイズ、色、再生設定などを数値指定できます。
- 配置ボタンで前面・背面、整列、等間隔配置、キャンバス中央配置ができます。

## フォルダ選択方法

「メディア」オブジェクトを追加し、右側プロパティの「フォルダを選択」を押します。選択したフォルダ内の画像・動画を検出し、ファイル名順またはランダムで繰り返し再生できます。サブフォルダを含める設定や手動更新にも対応しています。

## 表示開始方法

上部の「▶ 表示開始」を押すと編集UIを隠して表示モードへ切り替わります。表示モードでは16:9キャンバスを画面に合わせて等比率表示し、余白は黒背景になります。`Esc` キーで編集モードへ戻ります。

## 保存とバックアップ

- 編集内容はIndexedDBへ自動保存されます。
- 「保存」で手動保存できます。
- 「JSON出力」でバックアップを書き出せます。
- 「JSON読込」でバックアップを復元できます。
- JSONにはフォルダアクセス権限やローカルメディアファイルの実体は含まれません。

## ブラウザの権限制約

フォルダ選択はFile System Access APIの権限に依存します。ブラウザ再起動、端末変更、権限取り消しなどにより、再度フォルダ選択が必要になることがあります。ディレクトリハンドルはIndexedDBに保存しますが、権限自体をJSONへ移行することはできません。

## 音声付き動画の自動再生制限

ブラウザは音声付き動画の自動再生を制限する場合があります。そのため動画とメディアプレイヤーの既定値はミュートです。音声付きで再生したい場合は、ブラウザ設定や利用者操作の条件に注意してください。

## 別端末へ移行する場合の注意

JSONバックアップを移行先端末で読み込んだ後、画像・動画ファイルやメディアフォルダは移行先で再指定してください。ローカルファイルそのものやフォルダ権限はバックアップに含まれません。

## 既知の制約

- `file://` 実行を前提とするため、ブラウザや企業ポリシーによりFile System Access APIが制限される場合があります。
- 単一画像・単一動画のFileオブジェクトはブラウザセッションや保存仕様の制約を受けるため、長期運用ではメディアプレイヤーのフォルダ指定を推奨します。
- 回転したオブジェクトの当たり判定やリサイズは簡易実装です。
- 表示モード中のシーン切替はシーンの表示秒数に基づく簡易切替です。
- 外部通信、CDN、npm、サーバー処理は使用していません。
